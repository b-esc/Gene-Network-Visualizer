func PreviewGeneByUid ¶ 

func PreviewGeneByUid(w http.ResponseWriter, r *http.Request)

PreviewGeneByUid replicates QueryGeneByUid's behavior but only prepares the response data with genes to be displayed in a smaller table (no graph) This is called whenever 'more info' is selected by the user on the clientside

func QueryGeneByUid ¶ 

func QueryGeneByUid(w http.ResponseWriter, r *http.Request)

QueryGeneByUid is primarily a debugging tool for table + visualizer using the UIDs in the original gene_info csv takes a user's UID query, finds N most related genes in dataset (maxRes), filters out all edges not explicitly outgoing from root UID if isNhood = false returns an object containing properties: genes: array of Gene objects containing relevant edges links: array of edges to render (isNhood may impact this)
